## About
Todo app is an application that allows you to manage daily tasks in a simple way.

## Getting Started 
Clone respository:
```
git clone https://github.com/lucianoumc1/todoApp.git
```

Install the dependencies:
```
npm install
// or
yarn
```

Run in dev mode.
```
$ npm start
// or
$ yarn dev
```
